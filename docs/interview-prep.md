# 🎯 DSA & Interview Prep

Algorithm patterns, SQL practice, and interview platforms for AI/ML and Data Engineering roles.

## Algorithms & Data Structures

- **[AlgoMaster — DSA Patterns](https://algomaster.io/practice/dsa-patterns)** — Pattern-based practice (sliding window, two pointers, etc.) rather than random LeetCode grinding.
- **[LeetCode Top Interview 150](https://leetcode.com/studyplan/top-interview-150/)** — The canonical short list for MAANG/big-tech screens.
- **[Blind 75](https://www.educative.io/blog/blind-75-leetcode)** — Original curated list; still the gold standard for first-pass prep.
- **[Programming Street 150](https://github.com/Preparation-Street/Programming-Street-150)** — 150 problems with explanations.
- **[Grokking Algorithms (PDF)](https://edu.anarcho-copy.org/Algorithm/grokking-algorithms-illustrated-programmers-curious.pdf)** — Visual, friendly intro. Best read *before* grinding LeetCode.
- **[InterviewBit Time Complexity](https://www.interviewbit.com/courses/programming/time-complexity/)** — Big-O fundamentals if your CS background is shaky.
- **[RealDSA](https://realdsa.com/)** — Practical DSA framing.

## SQL

- **[LeetCode Top SQL 50](https://leetcode.com/studyplan/top-sql-50/)** — Essential for any data role.
- **[DataLemur](https://datalemur.com/questions)** — Real questions from FAANG data interviews; better quality than LeetCode SQL for analytics roles.
- **[StrataScratch](https://platform.stratascratch.com/coding?code_type=1)** — SQL + Python coding combined; closer to real DA/DS interview format.
- **[SQL Roadmap (roadmap.sh)](https://roadmap.sh/sql)** — Visual learning path from beginner to advanced.
- **[SQLZap — CTEs & Subqueries](https://sqlzap.com/lessons/subqueries-and-common-table-expressions)** — The two topics interviewers always ask about.
- **[SQL Interview Questions (Notion)](https://interviewquestions.notion.site/SQL-Interview-Questions-961bac970ff54c90a5bbd8ca614a4b8c)** — Comprehensive question bank with answers.
- **[Interview Query](https://www.interviewquery.com/)** — DS/DE-focused interview platform.

## Mock Interviews & Coaching

- **[Interviewing.io](https://interviewing.io/)** — Anonymous mock interviews with FAANG engineers; pay-after-you-land model available.
- **[Exponent](https://www.tryexponent.com/)** — Video-based PM/Eng prep; strong for behavioral and system design.
- **[Taro](https://www.jointaro.com/membership/)** — Career coaching from senior engineers at top companies.
