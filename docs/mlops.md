# 🚀 MLOps & Deployment

Infrastructure for serving, scaling, and orchestrating ML systems.

## Serverless & Cloud ML

- **[Modal](https://modal.com/)** — Serverless GPU/ML infrastructure. Probably the easiest way to deploy a Python function on a GPU.
- **[Replicate](https://replicate.com/)** — Run open-source models via API; good for prototyping before building your own infra.
- **[Prime Intellect](https://www.primeintellect.ai/)** — Distributed AI training platform.

## LLM Orchestration

- **[Portkey Gateway](https://github.com/Portkey-AI/gateway)** — LLM gateway with routing, fallbacks, observability across providers.

## Self-Hosted

- **[Coolify](https://coolify.io/)** — Self-hosted Heroku/Vercel alternative; good for running your own model APIs.
- **[Vespa](https://vespa.ai/)** — Vector search and ranking engine for production RAG.

## Collaboration

- **[Zerve](https://www.zerve.ai/)** — Collaborative data science workspace.
