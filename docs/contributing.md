# Contributing

The full contributing guide lives in the repo root: **[CONTRIBUTING.md](https://github.com/HarshMehta9000/ai-data-engineering-resources-cl/blob/main/CONTRIBUTING.md)**.

## Quick version

1. Find the right topic file in `docs/`.
2. Add your entry in this format:

   ```markdown
   - **[Name](URL)** — One-line "why". When would someone reach for this?
   ```

3. No duplicates. Search first.
4. Open a PR.

For non-PR suggestions, file an [issue](https://github.com/HarshMehta9000/ai-data-engineering-resources-cl/issues/new/choose).
