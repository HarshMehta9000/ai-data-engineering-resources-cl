# AI & Data Engineering Resources

> A curated knowledge base for AI engineers, ML practitioners, and data engineers.

Every link comes with a one-line "why" — when you'd reach for it over alternatives. No raw bookmark dump.

## How to use this site

- **Searching for something specific?** Hit the search bar (top right) — it indexes every resource and annotation.
- **Browsing?** Pick a topic from the sidebar.
- **Found a dead link?** [Open an issue](https://github.com/HarshMehta9000/ai-data-engineering-resources-cl/issues/new/choose).
- **Want to contribute?** See the [contributing guide](contributing.md).
