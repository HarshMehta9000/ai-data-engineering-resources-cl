# 📝 Blogs & Engineering Writing

Writers and publications worth subscribing to.

## Long-Standing Voices

- **[Paul Graham — Write Simply](https://paulgraham.com/simply.html)** — Short essay on writing; applies to engineering communication too.
- **[Simon Willison](https://simonwillison.net/)** — Daily-ish posts on LLMs, SQLite, Python tooling. ([example post on HTML tools](https://simonwillison.net/2025/Dec/10/html-tools/))
- **[Eli Bendersky](https://eli.thegreenplace.net/)** — Deep, careful technical writing on systems and languages. (His Raft series is featured in [system-design.md](system-design.md).)

## Industry Analysis

- **[SemiAnalysis](https://semianalysis.com/)** — Chip industry, GPU supply, training infrastructure. The reference for what's actually happening in compute.
- **[Stack Overflow Blog](https://stackoverflow.blog/)** — Engineering culture and craft. ([example](https://stackoverflow.blog/2026/04/09/gen-z-needs-a-knowledge-base-and-so-do-you/))

## ML Practitioners

- **[Pratham Prasoon](https://www.prathamprasoon.com/)**
- **[The Modern Software](https://themodernsoftware.dev/)**

## Performance / Frontend

- **[Frontend Performance Guide (Nehha)](https://hellonehha.medium.com/front-end-performance-guide-for-senior-developers-5e2b4b5ba7da)** — Concrete, senior-level perf advice.
