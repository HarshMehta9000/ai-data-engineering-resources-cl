# 🛠️ Developer Tools

Editors, terminals, and infra you use every day.

## Terminal & Editor

- **[Warp](https://www.warp.dev/)** — AI-native terminal. Worth trying even if you're a tmux/iTerm loyalist.

## Project Management

- **[Linear](https://linear.app/)** — Fast issue tracking; the speed difference vs. Jira is night and day.

## Communication

- **[Superhuman](https://superhuman.com/)** — Keyboard-first email; pricey but the productivity claims are real if you live in your inbox.

## Design

- **[Sketch](https://www.sketch.com/)** — Mac-native design tool.
- **[Paper Design](https://paper.design/)** — Newer design tool worth watching.

## Code Quality

- **[SonarQube](https://www.sonarsource.com/products/sonarqube/)** — Static analysis and code quality tracking.

## Git & Repo Hygiene

- **[BFG Repo-Cleaner](https://rtyley.github.io/bfg-repo-cleaner/)** — Faster than `git filter-branch` for removing secrets and large files from history.

## Web Archiving

- **[Archive.ph](https://archive.ph/)** — Permanent snapshots of web pages; useful for citing or surviving link rot.

## CV / Resume

- **[RenderCV](https://docs.rendercv.com/user_guide/)** — YAML-driven CV generator with version control.

## Free for Students

- **[GitHub Student Pack](https://education.github.com/pack)** — Free credits for ~100 dev tools while you're a student.
