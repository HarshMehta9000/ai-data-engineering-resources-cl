## 🌐 Browse Online

Available as a searchable [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) site at **https://HarshMehta9000.github.io/ai-data-engineering-resources-cl/** (deployed automatically from `main`).

To run locally:

```bash
pip install -r requirements-docs.txt
mkdocs serve
```

Then open <http://127.0.0.1:8000>.

## ✅ How Links Are Curated

Each entry has:

1. **A canonical URL** — the project's homepage or repo, not a random blog link.
2. **A one-line "why"** — when you'd reach for this resource over alternatives.
3. **No duplicates** — links appear in exactly one topic file.

Dead links are caught by [Lychee](https://github.com/lycheeverse/lychee-action) running on every push and weekly via cron. See the [link-check workflow](.github/workflows/links.yml).
